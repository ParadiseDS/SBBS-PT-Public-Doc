# SBBS PT 公共文档仓库

## 内容概览

仓库包含 SBBS PT 公开性的文档资料，主要用于存档和规则修订记录。

该文档中包括

+ 规则细则文档

+ 教程文档

+ 细则文档的种子简介和做种文件

## 修订须知

+ 修订总规则 / 常见问题时，直接修改相应 md 文件。尽量做到一条规则/问题一次提交。

+ 修改以种子形式存在的文档时，需要修改**根目录下的相应 txt** 以及 **种子简介中的相应文档**。

    做种文件不予修改，**除非种子被重新发布**。

    重新发布种子时，用根目录中的相应文档覆盖做种文件中的同名文档后，使用该文档重新制作种子，发布。
